# Dataset

This folder contains the dataset used for predicting customer churn.

## Description
- The dataset consists of customer demographic and usage-related features.
- Data preprocessing includes handling missing values, encoding categorical features, and addressing class imbalance.

## Target Variable
- **churn**: Indicates whether a customer has churned (1) or not (0).

## Note
The dataset is not included in this repository.  
You can replace it with any compatible customer churn dataset to train and evaluate the model.
